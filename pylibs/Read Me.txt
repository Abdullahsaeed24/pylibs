Pre-Requiests 

* Windows 10 Should be Installed on the Target Machine  
* The Internet Connection Must be stable


The " pylibs " Folder consists of the following files 

 1. download python and install.ps1 --> download python and install it using powershell cmd 
 2. libs setup.ps1 -->  download python libs and install them using powershell cmd 
 3. install-chrome.ps1 --> download google chrome and install it using powershell cmd 

 Now you know the structure of the " pylibs " folder
 
 
 
Pre-request :
 1. Write “powershell”in the search bar (left bottom side) of windows
 2. Choose the blue icon and open it as administrator
 3. Write this command   “ Set-ExecutionPolicy RemoteSigned” 
 4. Write Y and then close it 
	
please follow the next procedures to automate the setup process

Python installation:
 1. Download install folder from google drive or from flash drive. 
 2. Right click on the download_python_and_instal.ps1 in the pylibs folder and click to Run with PowerShell.
 3. Follow the instructions during installation.
 4. After installation right click on the libs_setup.ps1 and click to Run with PowerShell.
 5. After installation the PowerShell window will close automatically.

Google Chrome installation:
 1. Right click on the install-chrome.ps1 in the pylibs folder and click to Run with PowerShell.
 2. Follow the instructions during installation.
 3. After installation, close the PowerShell window.
